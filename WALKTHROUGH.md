# 校园活动报名与签到系统 - 运行指南

本文档包含了系统初始化、启动以及核心功能的完整演示流程。

## 1. PostgreSQL 初始化

首先确保您的机器上安装并运行了 PostgreSQL。

1.  **创建数据库**:
    ```bash
    psql -U postgres
    # 在 psql 终端中执行:
    CREATE DATABASE campus_db;
    ```
2.  **配置连接**:
    打开 `backend/src/main/resources/application.yml`，确认 `url`, `username`, `password` 与您的本地配置一致。
    ```yaml
    spring:
      datasource:
        url: jdbc:postgresql://localhost:5432/campus_db
        username: postgres # 请按需修改
        password: password # 请按需修改
    ```
3.  **注意**: Spring Boot 启动时会自动根据 Entity 类更新表结构（`ddl-auto: update`），因此无需手动创建表。

---

## 2. 启动方式

建议在两个独立的 Terminal 窗口中分别启动后端和前端。

### 后端 (Spring Boot)

```bash
cd backend
# 使用 Maven 包装器启动
./mvnw spring-boot:run
```

- 启动成功标志：控制台输出 `Started CampusApplication in ... seconds`。
- 同时会打印日志：`Initialized Admin Account: admin`。

### 前端 (Vue + Vite)

```bash
cd frontend
# 安装依赖 (如未安装)
npm install
# 启动开发服务器
npm run dev
```

- 启动成功标志：控制台输出 `Local: http://localhost:5173/`。

## 4. 完整“报名 → 签到”操作演示

本示例需要用到两个账号：**管理员 (`admin`)** 和 **普通用户**。建议使用**Chrome 普通窗口**作为管理员，**Chrome 无痕模式**作为普通用户来同时模拟。

### 步骤 A：管理员发布活动 (含签到配置)

1.  **管理员**登录，点击右上角菜单进入 **`/admin` (后台管理)**。
2.  点击 **“+ 发布新活动”**。
3.  **填写表单** (关键步骤):
    - 标题: `秋季校园马拉松`
    - 时间: 今天 08:00 ~ 今天 18:00
    - **签到窗口 (非常重要)**: 设置一个**包含当前时间**的时间段。
      - 例如当前是 14:00，则设为 13:50 ~ 14:10。
    - 人数: 0 (不限)。
4.  点击保存，列表中出现该活动。

### 步骤 B：用户报名

1.  **普通用户** (在无痕窗口) 访问 `/login`，点击“注册新账号”。
    - 注册: `student1` / `123456` / `张三`。
2.  登录后进入首页，点击`秋季校园马拉松`的 **“查看详情”**。
3.  点击底部的 **“立即报名”** 绿色按钮。
    - 系统提示“报名成功”。
    - 按钮区域变为“✅ 您已报名参加此活动”。

### 步骤 C：用户签到

1.  保持在详情页。由于当前时间处于管理员设置的“签到窗口”内，页面会显示 **黄色“立即签到”按钮**。
2.  点击 **“立即签到”**。
    - 系统提示“签到成功”。
    - 按钮区域更新为 **“🎉 已签到”**。
    - “取消报名”按钮消失。

---

## 5. 管理员查看签到名单

1.  回到 **管理员** 窗口。
2.  在后台管理页面的活动列表中，找到`秋季校园马拉松`，点击 **蓝色“名单”按钮**。
3.  **查看结果**：
    - 表格中应显示一行记录。
    - **姓名**: `张三`
    - **签到状态**: `已签到` (绿色加粗文本)。
    - **签到时间**: 显示刚才点击签到的具体时间 (如 `2023/10/24 14:05:00`)。

### 对比测试 (未签到用户)

1.  注册另一个用户 `student2` / `李四`。
2.  只点击“报名”，**不点击**“签到”。
3.  管理员刷新名单，将会看到：
    - 姓名: `李四`
    - 签到状态: `未签到` (黄色文本)。
    - 签到时间: `-`。
